◆ clonarMatriz()

void clonarMatriz ( Lista *  listaMatrizes)

Clona as matrizes presentes na memória, qualquer matriz pode ser clonada e seu id sempre será o próximo ID disponível.

O objetivo aqui é permitir que um usuario possa utilizar uma instância antiga da matriz para operações futuras.

clonarMatriz Clonar matriz presentes na memória

Parâmetros
listaMatrizesLista com todas as matrizes do sistema

O objetivo aqui é gerar uma matriz escalonada na forma escada de Gauss. Além da matriz ser modificada na forma de Gauss é exibido um vetor com as respostas do sistema de equação linear.

◆ operacaoMatrizComplexas()

Por padrão essas funções aqui estão sinalizadas com o nome de operações complexas, porém o que temos na verdade é um conjunto de metodos algebricos para solução de problemas comuns como é o caso de encontrar o kernel de uma matriz, encontrar a solução de equações lineares e afins.
