Descrição detalhada

Matriz n x m.

Este tipo define a estrutura matemática matriz. Uma matriz é um vetor de vetores. As matrizes definidas por esse tipo são bidimensionais aonde cada celula da matriz tem coodenadas i,j logo M(i,j) define uma celula da matriz M de tamanho n x m, sendo n o número de linhas e m o número de colunas. Esse tipo possui metodos auxiliares que auxiliam a execução de todas as operações básicas.
