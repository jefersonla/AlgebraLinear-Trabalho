◆ operacaoGaussMatriz()

Vetor* operacaoGaussMatriz ( Matriz *  matriz)

Realiza o metodo de Gauss para encontrar as raizes de um sistema de equações.

Essa função assume que a matriz contém ao menos duas colunas e a última é sempre o resultado. O retorno dessa função é um vetor de doubles com a dimensão do número de colunas - 1 de todas as raizes dessa equação, caso o resultado não seja possível o resultado é nulo e caso existam múltiplas sóluções o retorno é nulo também. Esses casos podem ser melhor tratados no futuro. O procedimento para execução do metodo de Gauss é o metodo de Gauss com pivotamento a fim de evitar divisões por zero.
