Multiplica uma linha por um escalar qualquer.

Esta operação dada uma linha da matriz multiplica a mesma por um escalar que pode ser positivo ou negativo. Este escalar tem que ser diferente de 0. Formato L1 <- L1 * E .

operacaoMultiplicaPorEscalar Multiplica uma linha por um escalar

Parâmetros
matrizMatriz que receberá a operação
linhaNumero da linha que receberá a operação
escalarNúmero diferente de 0 a qual as celulas da linha receberão a operação
Retorna
Retorna true se a operação não teve erros e falso caso contrário

Soma uma linha a outra multiplicada por um escalar.

Esta operação aplica uma soma de uma linha com outra, com a outra linha podendo ser modificada por um escalar no formato L1 <- L1 + E * L2. É importante lembrar que L1 nesse formato não pode ser modificado nesta operação e somente L2 será multiplicado pelo escalar. O escalar não pode ser igual a 0, porém pode ser um número negativo ou uma fração.

operacaoSomaEntreLinhas Soma uma linha a outra multiplicada por um escalar

Parâmetros
matrizMatriz que irá receber a operação
linha_resLinha do resultado
linha_inLinha que será somada a linha que irá receber o resultado
escalarEscalar que irá ser multiplicado pela outra linha
Retorna
Retorna true se a operação ocorreu bem, e falso caso contrário

Troca uma linha da matriz com outra da mesma.

Esta operação elementar apenas trocar o valor das linhas, não é possível modificar os valores das mesmas nesta operação. Formato L1 <-> L2 .

Realiza o produto entre duas matrizes.

O produto entre duas matrizes é dado pela multiplicação das linhas de uma matriz com a coluna da outra. No produto de matrizes a matriz resultante tem formato A (n x k) <- M1 (n x m) . M2 (m x k)

produtoMatriz Produto de matrizes

Parâmetros
matriz1Matriz 1
matriz2Matriz 2
Retorna
Retorna uma nova matriz, resultado do produto entre as matrizes

< Define uma matriz vazia, ou seja com 0 em todos os valores
